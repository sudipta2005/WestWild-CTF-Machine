# WestWild Capture The Flag (CTF) Machine Development

## Project Overview

This project demonstrates the development and customization of the **WestWild** Capture The Flag (CTF) machine using **Kali Linux** as the attacker machine and **Ubuntu 14.04.6 LTS** as the target machine.

The project includes reconnaissance, service enumeration, SMB exploitation, credential discovery, SSH access, privilege escalation, Apache web server configuration, website customization, user management, hostname modification, and custom flag creation.

---

## Project Information

- **Project Title:** Capture The Flag (CTF) Machine Development
- **Target Machine:** WestWild
- **Attacker Machine:** Kali Linux
- **Operating System:** Ubuntu 14.04.6 LTS
- **Virtualization Platform:** VMware Workstation 17 Pro

- Configure and customize a Linux CTF machine.
